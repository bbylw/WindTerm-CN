Features

全方位终端解决方案

为专业开发者和运维工程师精心打造，覆盖 SSH 连接、文件传输、会话管理到安全防护的完整工作流

SSH 全功能支持

核心协议

完整的 SSH v2 协议实现，支持密码、公钥、键盘交互和 GSSAPI 多种认证方式。内置 SSH Agent 和代理转发，X11 转发可直接运行远程 GUI 程序。

ProxyCommand ProxyJump ControlMaster 端口转发 X11

极速文件传输

SFTP / SCP

集成 SFTP/SCP 客户端和本地文件管理器，高速传输模式下上传速度可达 247 MB/s，远超 FileZilla 和 WinSCP。支持 XModem、YModem、ZModem 传统传输协议。

下载/上传 重命名 批量操作 断点续传

多协议接入

Telnet、Raw TCP、Serial 串口全覆盖，支持 HTTP/SOCKS5 代理和跳板机代理，满足各类网络环境需求

专业终端仿真

完整支持 vt100 至 vt520 以及 xterm-256-colors，Unicode 13、Emoji、真彩色、鼠标协议一应俱全

跨平台 Shell

Windows Cmd/PowerShell（含管理员），Linux/macOS 的 bash、zsh、PowerShell Core，原生 Tmux 集成

现代化界面

智能自动补全、命令面板、分屏视图、多主题配色、Powerline 美化、Vim 键绑定的双模切换

会话管理

会话树管理与对话框、自动/手动日志、重启恢复布局、同步输入到多个终端、启动时加载指定会话

安全增强

增强的用户名密码保护机制、锁定屏幕、SSH Agent 代理、粘贴确认对话框，保障运维安全
