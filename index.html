FEATURES

功能特性

为专业开发者和运维工程师设计的全方位终端解决方案

SSH 全功能支持

SSH v2 协议，支持自动登录、代理转发、X11 转发、端口转发（本地/远程/动态）、ProxyCommand/ProxyJump、ControlMaster

文件传输

集成 SFTP/SCP 客户端和本地文件管理器，支持 XModem/YModem/ZModem，高速传输可达 247 MB/s

多协议支持

Telnet、Raw TCP、Serial 串口协议，支持 HTTP/SOCKS5 代理和跳板机代理

终端仿真

支持 vt100/vt220/vt340/vt420/vt520/xterm/xterm-256-colors，Unicode 13、emoji、真彩色

跨平台 Shell

Windows Cmd/PowerShell（含管理员），Linux/macOS bash/zsh/PowerShell Core，Tmux 集成

现代化界面

自动补全、命令面板、分屏视图、多主题、Powerline 支持、Vim 键绑定模式

会话管理

会话树管理、自动/手动日志记录、重启恢复会话和布局、同步输入多终端

安全增强

增强的用户名密码保护、锁定屏幕、SSH Agent、GSSAPI 认证、粘贴确认对话框

极致性能

动态内存压缩减少 20%~90% 内存，无限回滚行数，千万行数据仅需 133MB 内存
